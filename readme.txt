Quickorder is a compact plugin that allows you to change the order of all your pages in one fell swoop.

== Description ==

Quickorder provides the ability to rapidly reorder pages, as of now reordering pages is very tedious, but the Quickorder plugin
grants you the ability to change all of your pages at once.

== Installation ==

1. Upload `quickorder.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on Order pages within the Manage tab


== Screenshots ==

1. Each page is listed out to the left under Page Name and their current Order beside it.

2. Select the page you would like an enter a value. the number 1 will show up in the top of the order.




== Frequently Asked Questions ==

= If I set multiple pages to the same number what will happen? =

Wordpress will sort them alphabetically.

= What if I set a page to a value of 0? =

If will be thrown out of the loop and be placed at the top.
